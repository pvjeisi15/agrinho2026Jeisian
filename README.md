# agrinho2026Jeisian
preciso de um codigo html e style para o agrinho 2026 com o tema Painel do Clima e Alerta de Geada: O site simula a recepção de dados climáticos das regiões do Paraná (Norte, Oeste, Sul). Ele emite um pop-up visual de alerta vermelho caso os termômetros simulem temperaturas próximas de uma geada negra, ajudando na proteção rápida das lavouras de café ou hortaliças.
